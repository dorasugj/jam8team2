# チーム名
OREGON\_TRAIL


# ゲームタイトル
飴ェー車　de ファイト！


# ゲーム概要
雨でできたアメ車（アメリカ車）をぶつけってあって、  
相手をおとす、おはじきゲーム

# プラットフォーム
パソコン（Windows）


# ゲーム内容
四人対戦でキーボード全体を使った（P1:Q, P2:Z P1:M P1:O）を使って順番に弾きあい、  
最後まで生き残ったら勝利。  
車は自動で回転しているので自分の進みたい方向を向いた瞬間にキーを押す。


# クレジット

- 植田弘一
- 堀井陽平
- 奥智弘
